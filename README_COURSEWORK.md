# Курсова робота - Інструкція

## Згенеровані файли

1. **Курсова_Робота_HabitTracker.docx** - Основний документ курсової роботи у форматі Word
2. **generate_coursework.py** - Python скрипт для генерації документу
3. **ПОЯСНЮВАЛЬНА_ЗАПИСКА.md** - Markdown версія (для редагування)

## Структура документу

Документ створено згідно методичних вказівок з файлу `met` і містить:

- Титульна сторінка
- Завдання на курсову роботу  
- Зміст
- Анотація (українською та англійською)
- Вступ
- Розділ 1: Аналітичний розділ
- Розділ 2: Алгоритмічне забезпечення
- Розділ 3: Проектування ІС
- Розділ 4: Реалізація ІС
- Розділ 5: Забезпечення якості
- Висновки
- Перелік посилань

## Формат

- Шрифт: Times New Roman, 14pt
- Міжрядковий інтервал: 1.5
- Вирівнювання: по ширині
- Відступ першого рядка: 1.27 см (0.5 дюйма)

## Як додати більше контенту

Щоб розширити документ до 70+ сторінок:

1. Відредагуйте `generate_coursework.py`
2. Додайте більше розділів/підрозділів
3. Запустіть: `python3 generate_coursework.py`

## Тема проекту

**Розробка веб-системи для відстеження звичок та досягнень (Habit Tracker)**

Технології:
- Backend: Spring Boot 3.2, Java 17, MongoDB
- Frontend: React 18, Vite, Tailwind CSS
- Авторизація: JWT
- Архітектура: REST API, SPA
